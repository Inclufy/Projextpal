## Domain violations (highest priority)
1. **Governance board makes NO gate decisions** ‚Äî `PMIGovernanceBoard.decisions` is a
   free JSON list (`models.py:61`), with **no FK to a component** and **no `@action`**
   to authorize/continue/stop. A board record cannot change a component's `status`.
   Test only asserts `POST ‚Üí 201` (`tests/test_governance.py:19-20`): pure storage. ‚ùå
2. **No program charter** ‚Äî no model/field tracing the program to organizational
   strategy or business case anywhere in `pmi/models.py`. ‚ùå
3. **No benefits management** ‚Äî no benefits identify‚Üíplan‚Üídeliver‚Üítransition‚Üísustain in
   PMI. (A *separate* `programs.ProgramBenefit` exists at `programs/models.py:164` with
   a `realized` status, but it is NOT PMI's and has no transition/sustain lifecycle.) ‚ùå
4. **No stakeholder engagement** ‚Äî no stakeholder register / power-interest / engagement
   plan in PMI. ‚ùå
5. **No program roadmap / life-cycle phases** ‚Äî components carry only flat
   `proposed/approved/active/completed/cancelled` (`models.py:15-21`); no Definition‚Üí
   Delivery‚ÜíClosure phase, no roadmap entity. ‚öÝÔ∏è (status stored, not governed)
6. **Status transitions unguarded** ‚Äî any client can PATCH a component to any status
   via the generic `ModelViewSet` (`views.py:22`); no gate, no authorization record. ‚ùå
7. **Stranded module** ‚Äî sidebar PMI case links `/charter`, `/benefits`, `/kpis`,
   `/stakeholders`, `/milestones`, `/governance` (`AppSidebar.tsx:149-191`) but those

## Feature proposals + redesign (ranked ‚Äî both directions)
**FLAGSHIP REDESIGN ‚Äî Governance Gate Decision over components (weight√óvalue√ólow-effort,
top priority).** Add `PMIGateDecision(board FK ‚Üí PMIGovernanceBoard, component FK ‚Üí
PMIComponent, decision: authorize/continue/hold/stop, gate: definition/delivery/closure,
rationale, decided_at)` + a viewset `@action` `authorize_component` / `stop_component`

status). Screen: Governance page ‚Üí per-component gate panel. Cite: `models.py:45`,
`views.py:22`. This converts the board from a minutes log into a governing body.
